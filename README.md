# 🚪MysteriousRooms

2D自動横スクロール型のシューティングゲームです。
敵機を倒してスコアを稼ぎ、より高いスコアを目指しながらゴールへ向かうのが目標のゲームです。
敵に当たらないように敵機を倒していき、最後に現れるボスを倒せばゲームクリアです。

※このリポジトリは学習/ポートフォリオ公開を目的としており、一部の外部アセットはライセンス上の理由により除外しております。
